In short

Use the topic posts for depth and narrative. Use the table below (or my homepage badge grid) when you only need dates, issuers, and official links.

What is Credly?

Credly hosts digital badges tied to a public page per credential: issuer, issue date, and a stable URL you can share with recruiters or clients.

My homepage loads the same badge set from a small JSON file in the repo so the grid stays aligned with Credly.
