What I learned: Cloud Security Foundations

How AWS Academy Cloud Security Foundations strengthened the security lens I use in platform and DevOps work.

In short

A practical base layer for securing accounts, networks, data, and detective controls on AWS—before diving into vendor-specific hardening guides.

The credential

AWS Academy Graduate — Cloud Security Foundations — Training Badge. Verify on Credly.

What the course stresses

Security foundations tie identity, logging, encryption, and network boundaries into habits you can apply across services—so you are not learning controls in isolation.

Ideas and skills

  • Shared responsibility — what AWS secures vs what your team must configure.
  • IAM discipline — roles, policies, and least privilege as daily design inputs.
  • Detective controls — why logs and alerts only help when someone owns the response.
  • Data protection — encryption in transit and at rest as default questions, not extras.

How I use this

This material pairs naturally with IAM policy JSON anatomy (field-by-field guide), Cloud Architecting, and with governance study in ISO/IEC 27001 Lead Auditor when discussing ISMS evidence and risk.
